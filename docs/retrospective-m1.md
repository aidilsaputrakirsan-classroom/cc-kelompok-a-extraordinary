# Retrospective — Milestone 1

## 🟢 Apa yang Berjalan Baik?
- **Docker Compose:** Setup environment backend, frontend, dan database berhasil berjalan dengan satu perintah `docker compose up`.
- **Kolaborasi Awal:** Pembagian tugas di awal milestone cukup jelas sehingga setiap anggota tahu area mana yang harus dikerjakan.
- **Progress UI:** Tampilan frontend sudah mulai merepresentasikan fitur utama yang direncanakan.

## 🔴 Apa yang Perlu Diperbaiki?
- **Komunikasi Tim:** Masih sering terjadi pengerjaan double atau kebingungan status task karena koordinasi yang kurang intens.
- **Sinkronisasi Kode:** Sering terjadi conflict saat push ke main karena semua orang bekerja di branch yang sama (main).
- **Dokumentasi:** Dokumentasi API dan setup guide sering terlambat di-update setelah ada perubahan kode.

## 🔵 Action Items untuk Milestone 2
- **GitHub Flow:** Wajib menggunakan feature branch dan Pull Request (PR) untuk setiap perubahan kode, sesuai dengan Modul 9.
- **Daily Sync:** Melakukan koordinasi singkat setiap hari untuk sinkronisasi status task.
- **Code Review:** Melakukan review terhadap PR teman sebelum di-merge ke branch utama untuk menjaga kualitas kode.
