## 🚀 Cara Menjalankan Project

### 1️⃣ Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2️⃣ Frontend 
```bash
cd frontend
npm install
npm run dev
```
